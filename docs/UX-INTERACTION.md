# Task Horizon UX 交互文档

版本基线：`plugin.json` 2.3.7  
文档目的：给前端重建界面时使用，描述页面结构、关键组件、交互状态、响应式规则和边界状态。

## 1. 设计原则

- 第一屏就是工具本体，不做营销页。
- 功能密度高但层次清晰：顶部做全局操作，视图内做上下文操作，详情承载深度编辑。
- 图标按钮必须有 tooltip；陌生图标需有可读名称。
- 任务列表要便于扫描：标题、状态、日期、文档、重要性、进度优先。
- 高频操作尽量不弹大窗：状态、日期、重要性、时长、备注走 popover 或 inline edit。
- 移动端优先保证“查看、勾选、改日期、新建、搜索、切换视图”这些核心动作。
- 任何刷新和保存都不能粗暴打断正在输入、拖拽或选择的用户。

## 2. 全局布局

### 2.1 桌面页签

结构：

1. 顶部工具栏：主页、新建、分组、规则、搜索、视图、刷新、设置、菜单。
2. 文档页签区：全部文档或分组内文档列表，可点击切换。
3. 主内容区：随当前视图切换。
4. 可选右侧区：清单/看板详情、日历侧栏、白板任务池。
5. 浮动层：多选批量条、日历拖拽浮层、时间轴浮动工具、弹出菜单。

布局要求：

- 顶部工具栏高度固定，内容区撑满剩余高度。
- 文档页签过多时横向滚动或折叠菜单。
- 桌面窄宽度下，文档分组下拉仍必须可见。
- 不要把主工作区塞进装饰性卡片。

### 2.2 Dock

Dock 侧栏空间窄：

- 默认视图可跟随移动端设置。
- 视图切换可放到底部或紧凑菜单。
- 清单紧凑模式优先。
- 时间轴工具条使用浮动按钮组。
- 右侧详情应改为抽屉或替换式面板，避免双栏挤压。

### 2.3 移动端

移动端入口：

- 文档顶栏按钮短按：快速新建任务。
- 文档顶栏按钮长按：打开任务管理器。
- 设置中可对调长短按。

移动端任务管理器：

- 顶部保留返回/关闭、分组、搜索、刷新等最小操作。
- 视图切换放到底部导航或菜单。
- 任务详情使用底部抽屉。
- 日历长按 1 秒触发新建或拖拽。
- 浮动操作条要避开安全区 `env(safe-area-inset-bottom)`。

## 3. 导航与入口

### 3.1 主入口

- 点击思源顶栏图标：打开/聚焦 Task Horizon 页签。
- 点击文档顶栏图标：按设置执行打开管理器或快速新建。
- 从文档块菜单：可将当前文档加入分组。
- 从任务右键菜单：打开详情、删除、跳转、设置字段等。

### 3.2 主页入口

- 顶栏左侧主页图标切换到主页总览。
- 主页打开时隐藏普通任务视图主体。
- 再次选择任一任务视图退出主页。

### 3.3 视图切换

视图项：

- 清单
- 表格
- 时间轴
- 看板
- 日历
- 白板

行为：

- 当前视图高亮。
- 未启用的视图不显示。
- 切换视图时保持当前分组、规则、搜索条件和文档页签。
- 日历/白板等重型视图切换时需要显示轻量加载态。

## 4. 顶部工具栏交互

### 4.1 新建任务

入口：顶部加号、标题分组加号、看板右上角、白板文档卡片加号、移动端快速新建。

表单字段：

- 任务内容，必填。
- 目标文档，可选；未选时跟随当前文档、当前分组默认文档或全局默认文档。
- 重要性。
- 状态。
- 开始日期。
- 截止日期。
- 是否置顶，默认由设置控制。

成功后：

- 插入思源文档。
- 当前视图局部刷新。
- 若在某个标题分组内新建，任务写入对应标题下。

### 4.2 文档分组

下拉内容：

- 全部文档。
- 自定义分组。
- 笔记本分组。

行为：

- 选择分组后重新加载该范围任务。
- 下拉打开后延迟检查远端分组设置变更。
- 分组不存在时回落到全部文档。

### 4.3 规则

规则下拉：

- 无规则/全部。
- 用户创建的筛选规则。

行为：

- 选择规则后过滤当前任务集合。
- 规则可以影响自定义字段加载计划，因此切换后需要刷新必要字段。

### 4.4 搜索

行为：

- 输入关键词即时过滤当前任务集合。
- 不改变底层文档范围。
- 清空搜索恢复当前规则下的完整列表。

### 4.5 刷新

行为：

- 重新加载当前分组任务。
- 保留当前分组、规则、视图、折叠状态和详情选择。
- 如果开启伺服同步设置，刷新前先保存本地设置，再重载共享配置、任务补充元数据、白板数据和语义识别记录。

## 5. 主页总览

模块：

- 头部：标题、当前范围、时间范围切换。
- 概览卡：完成率、总数、未完成、已完成、逾期。
- 趋势卡：完成趋势折线/面积图。
- 热力图：按日期显示完成活跃度。
- 文档分布：各文档任务量和占比。
- 最近完成：任务标题、文档、完成日期。
- 逾期风险：任务标题、文档、截止日期、风险标签。

交互：

- 时间范围按钮切换统计窗口。
- 列表项点击打开任务详情或跳转任务。
- 图表 hover 显示日期和值。

空态：

- 当前范围无任务：显示“暂无任务统计”，提供“去添加文档分组”和“新建任务”入口。

## 6. 清单视图

### 6.1 行结构

任务行包含：

- 完成复选框。
- 子任务展开/折叠按钮。
- 标题文本。
- 循环任务图标/循环实例标识。
- 附件标识。
- 右侧紧凑字段：状态、重要性、开始日期、截止日期、时长、耗时、备注等，由设置控制。
- 当前选中/多选状态。

### 6.2 分组行

分组类型：

- 置顶。
- 文档。
- 标题。
- 任务名。
- 时间。
- 四象限。
- 普通未分组。

分组行包含：

- 展开/折叠箭头。
- 图标或颜色标签。
- 分组名称。
- 数量 badge。
- 时长汇总 badge。
- 标题分组下的新建任务按钮。

### 6.3 点击规则

- 点击复选框：切换完成状态。
- 点击行空白/主体：打开详情。
- 点击任务标题：按当前策略跳转原文档或打开详情。
- 右键/长按任务：打开任务上下文菜单。
- 点击展开箭头：折叠/展开子任务。
- 点击分组行：折叠/展开分组。

### 6.4 紧凑模式

要求：

- 子任务缩进必须清楚。
- 开启层级线时，显示纵向引导线。
- 有附件任务显示附件小图标。
- 右侧字段字号支持大/中/小。
- 字段过多时允许横向或换行，但不可压垮标题。

## 7. 表格视图

列交互：

- 表头可展示字段名。
- 列宽可拖拽。
- 列显示、顺序和默认宽度由设置控制。
- 单元格点击进入编辑状态。
- 日期字段使用日期选择器。
- 状态和重要性使用下拉选择器。
- 附件列显示数量和管理入口。

横向滚动：

- 表格主体横向滚动时，底部自定义滚动条同步。
- 固定高度内纵向滚动。

空态：

- 无任务：表格体中居中显示“暂无任务”。

## 8. 任务详情

### 8.1 展示形态

- 桌面清单：右侧嵌入详情卡。
- 看板：右侧详情面板。
- 移动端/白板：底部抽屉。
- 独立弹窗：用于部分上下文。

### 8.2 结构

头部：

- 位置 chips：文档、标题、跳转。
- 更多按钮。
- 保存按钮。
- 关闭按钮。

主体：

- 标题 textarea。
- 重要性按钮和菜单。
- 状态按钮和菜单。
- 开始日期、截止日期、时长、耗时、提醒、循环 chips。
- 子任务区。
- 附件区。
- 备注/自定义文本区。
- 自定义单选/多选字段区。

### 8.3 字段行为

- 标题自动高度，支持多行。
- 重要性菜单只显示图标/旗标样式，当前选中高亮。
- 状态菜单显示状态标签、颜色和选中勾。
- 日期 chip 点击弹出日期/时间选择。
- 时长 chip 点击弹出预设和自定义输入。
- 提醒 chip 点击打开番茄钟提醒设置；未启用番茄钟时隐藏。
- 循环 chip 点击打开循环规则设置。
- 附件支持添加、打开、删除、折叠/展开。
- 粘贴剪贴板内容时，可识别附件或文档 ID。

### 8.4 子任务

- 显示完成数/总数。
- 可新建子任务。
- 子任务可勾选完成。
- 点击子任务标题进入该子任务详情。
- 保存过程中保持输入区存在，并显示进行中状态。

### 8.5 关闭规则

- 点击关闭按钮关闭。
- 点击遮罩关闭时，需要确认没有正在输入的关键弹层。
- 鼠标拖拽出窗口外不能触发误关闭。

## 9. 批量操作

入口：

- 顶部菜单开启“多选模式”。
- 支持视图：清单、表格、时间轴。

交互：

- 开启后任务行点击切换选中。
- 底部浮动条显示已选数量。
- 无选中任务时批量动作禁用。
- 操作项：批量设置开始日期、截止日期、重要性、状态、更多自定义字段。
- 清空：取消当前选择但保留多选模式。
- 退出：关闭多选模式并清空选择。

响应式：

- 移动端底部浮动条避开视图切换栏和安全区。
- 时间轴浮动工具和多选条同时存在时，多选条位置上移。

## 10. 时间轴视图

布局：

- 左侧任务表格。
- 中间分割条。
- 右侧 Gantt 区。

工具：

- 隐藏/展开侧栏。
- 缩小。
- 放大。
- 适配范围。
- 定位今天。

交互：

- 拖拽分割条调整左侧宽度。
- 拖拽内容列边界调整任务内容列宽。
- 点击日期单元格编辑日期。
- 点击任务行打开详情。
- Gantt 区随日期字段变化刷新。

移动/Dock：

- 工具按钮进入浮动工具条。
- 横屏移动端可放入顶部紧凑栏。

## 11. 看板视图

布局：

- 横向列容器。
- 每列有标题、数量、状态颜色。
- 卡片列表。
- 可选右侧详情。

模式：

- 状态看板：按自定义状态列。
- 标题看板：按文档标题或标题分组，可显示已完成列。

卡片：

- 标题。
- 完成复选框。
- 子任务数量/进度。
- 重要性、状态、日期等字段 chips。
- 文档/标题信息。
- 循环标识、附件标识。

交互：

- 拖拽卡片到状态列，更新状态。
- 点击卡片打开详情。
- 右键打开任务菜单。
- 卡片字段按钮直接编辑对应字段。
- 新建按钮在当前看板上下文创建任务。

## 12. 日历视图

### 12.1 主日历

视图：

- 月。
- 周。
- 日。
- 三日。
- 其他 FullCalendar 支持的紧凑变体。

工具栏：

- 上一段/下一段。
- 今天。
- 视图切换 segmented control 或下拉。
- 标题显示当前日期范围。
- 侧栏开关。

事件类型：

- 任务日期事件。
- 日程事件。
- 番茄钟事件：专注、休息、计时、空闲。
- 中国节假日。
- 农历标签。

### 12.2 拖拽与编辑

- 拖拽任务到日历日期：设置任务截止日期。
- 拖拽到时间格：设置排期时间。
- 拖拽日程：移动日程开始/结束。
- 调整日程长度：更新结束时间。
- 拖拽任务时显示浮动重要性目标，放下后快捷设置重要性。
- 移动端长按后进入拖拽/新建。

### 12.3 侧栏

内容：

- 迷你日历。
- 我的日历列表。
- 文档/分组列表。
- 番茄钟来源开关。
- 未排期任务列表。

行为：

- 侧栏可折叠。
- 宽度可调整。
- 文档颜色可用于日程或任务日期颜色。

### 12.4 提醒

- 全天日程可配置默认提醒时间。
- 任务日期可配置提醒。
- 移动端支持原生通知注册。
- 过期提醒需要有聚焦/突出状态。

## 13. 白板视图

### 13.1 画布模式

工具：

- 平移。
- 选择/框选。
- 连接。
- 便签。
- 缩放。
- 重置视图。

对象：

- 任务卡片。
- 文档框。
- 便签。
- 连接线。

交互：

- 拖拽卡片改变位置。
- 框选多个卡片。
- 拖拽连接点建立任务关系。
- 双击/点击便签编辑。
- 右键任务卡打开菜单。
- 点击任务卡打开详情。
- 画布平移和缩放状态持久化。

### 13.2 全部页签卡片流

布局：

- 按文档分成多列瀑布流。
- 每个文档卡片包含标题、任务数量、新建按钮、拖拽手柄。
- 文档内按标题分组，支持折叠。
- 子任务树在文档卡片内缩进展示。

交互：

- 拖拽文档卡片调整顺序。
- 点击文档标题打开文档。
- 点击新建按钮在该文档中新建任务。
- 点击任务打开详情。
- 任务支持拖拽、右键、完成勾选。

移动端：

- 可配置单列或双列。
- 文档数量 badge 在双列时可隐藏以节省空间。

## 14. quickbar

触发：

- 文档内点击任务块。
- 可通过设置启用/关闭。

布局：

- 主行：状态、重要性、开始日期、截止日期、时长、备注、AI、提醒、更多。
- 备注编辑时可展开第二行工具栏。
- 字段可根据 `quickbarVisibleItems` 配置显示。

字段：

- 状态：下拉状态列表。
- 重要性：高/中/低/无。
- 日期：日期选择器。
- 时长：预设按钮 + 自定义输入。
- 备注：输入框/textarea + Markdown 工具。

行为：

- 失焦或确认后保存到块属性。
- 保存后通知插件主视图局部刷新。
- AI 关闭时隐藏 AI 优化入口。
- 提醒仅在番茄钟联动可用时显示。

## 15. 设置中心

### 15.1 布局

- 模态窗口。
- 左侧一级导航。
- 主区域内容。
- 常规设置内有横向/粘性子导航。
- 底部保存/取消按钮。

一级导航：

- 文档分组。
- 常规设置。
- 外观。
- 日历。
- AI。
- 规则管理。
- 四象限。
- 优先级算法。
- 关于。

### 15.2 文档分组

组件：

- 当前分组选择。
- 新建/重命名/删除分组。
- 添加笔记本。
- 手动输入文档 ID。
- 包含子文档开关。
- 当前分组文档列表。
- 排除文档列表。
- 导入滴答 CSV。
- 导出任务。

状态：

- 空文档列表显示“暂无文档，请添加”。
- 递归/笔记本来源展示标签。
- 分组为全部文档时，排除文档说明要区分“只影响全部文档视图”。

### 15.3 常规设置

分区：

- 视图与布局。
- 搜索与分组。
- 顶栏入口。
- 任务悬浮条。
- 番茄钟联动。
- 数据导入。
- 文档分组与管理。
- 状态选项。
- 自定义字段。

控件规范：

- 开关使用 switch。
- 数值使用 number input 或 slider。
- 模式选择使用 select/segmented。
- 多选字段配置使用 checkbox list。

### 15.4 外观

功能：

- 主题预设。
- TweakCN/shadcn CSS 变量导入。
- 移除导入主题。
- 列显示/排序/宽度。
- 重要性图标样式：Jira/旗标等。
- 分组颜色、进度条颜色、日历颜色微调。

列设置交互：

- 每列一行：显示开关、列名、上移、下移、宽度输入/滑块。
- 隐藏列可从列表中重新显示。
- 自定义列与内置列同等参与排序。

### 15.5 规则管理

列表态：

- 规则名称。
- 条件摘要。
- 编辑。
- 删除。
- 新建规则。

编辑态：

- 规则名称。
- 条件组。
- 排序规则。
- 字段选择支持内置字段和非文本自定义字段。
- 保存/取消。

### 15.6 AI 设置

字段：

- 启用 AI。
- 供应商：MiniMax / DeepSeek。
- API Key。
- Base URL。
- 模型。
- 温度。
- 最大 Token。
- 超时。
- 默认上下文：附近上下文 / 全文。
- 排期时间段，每行一段。
- 测试连接。

行为：

- 切换供应商时分别记忆各自配置。
- 关闭 AI 后主界面、quickbar 和菜单中 AI 入口都隐藏。

## 16. 空态、加载、错误

空态：

- 未配置文档：引导进入文档分组设置。
- 当前规则无结果：提示调整规则/搜索。
- 当前文档无任务：提供新建任务。
- 日历无事件：保留网格，不显示大面积错误。
- 白板无任务：提示从任务池或新建任务开始。

加载：

- 分组切换、刷新、日历初始化、白板初始化需要局部 loading。
- 加载不应覆盖正在编辑的详情字段，必要时延迟刷新。

错误：

- API 失败：toast/hint 显示失败原因。
- 任务不存在：提示“未找到任务”，并刷新当前视图。
- AI 失败：展示连接/配置/超时原因。
- 附件打开失败：提示可跳转到原文档检查。

## 17. 响应式规则

- 宽屏桌面：双栏/多栏布局，详情右侧常驻。
- 窄屏桌面：工具栏折叠次要按钮，文档分组下拉保留。
- Dock：优先清单紧凑，详情抽屉化，时间轴工具浮动。
- 手机竖屏：底部视图栏 + 抽屉详情，表格降级为横向滚动。
- 手机横屏：时间轴可使用顶部紧凑工具栏。
- 所有浮层需要避开系统安全区和底部工具条。
